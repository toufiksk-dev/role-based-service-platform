# 🚀 Role-Based Service Management Platform

A **production-ready full-stack MERN application** built with **MongoDB, Express.js, React.js, and Node.js**. The platform provides a secure role-based ecosystem for **Admin, Master Distributor, Distributor, and Retailer**, featuring dynamic KYC verification, wallet management, recharge services, transaction tracking, and hierarchical user management.

---

# 📌 Overview

This project is designed for digital service providers requiring a secure and scalable multi-user platform. It supports complete onboarding, KYC verification, wallet-based transactions, and role-specific dashboards, enabling efficient management across multiple user levels.

The application follows a modular architecture with secure authentication, REST APIs, and responsive user interfaces suitable for production environments.

## 👥 Multi-Role System

Supports four different user roles:

* **Admin**
* **Master Distributor**
* **Distributor**
* **Retailer**

Each role has dedicated permissions, dashboards, and management capabilities.

---

## 📄 Dynamic KYC Management

The Admin can create dynamic KYC forms for retailer verification.

Example fields:

* Full Name
* Mobile Number
* Aadhaar Card
* PAN Card
* Passport Photo
* Address Proof
* Bank Details
* Custom Document Fields

Retailers complete KYC online, and the Admin reviews, approves, or rejects submissions.
